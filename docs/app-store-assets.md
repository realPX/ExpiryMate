# 到期提醒管家 - 上架素材草案

## 品牌方向

- 关键词：到期、提醒、清晰、安心、轻量
- 图标方向：日历卡片 + 勾选，表达“到期事项被及时处理”
- 启动页方向：品牌蓝背景 + 居中图形，简洁、稳定、不花哨
- 截图风格：统一使用红橙蓝渐变风险语言，突出“先看风险，再做处理”

## 截图文案

### 1. 首页总览

- 标题：到期事项，一眼看清
- 副标题：已过期、今天到期、7 天内事项集中展示，优先级马上明确

### 2. 风险分级

- 标题：高风险事项，优先处理
- 副标题：红橙蓝分级提示，帮助你快速判断哪些内容最急

### 3. 列表管理

- 标题：筛选、搜索、批量整理
- 副标题：支持分类查看、归档恢复和批量操作，管理更省心

### 4. 长按速操作

- 标题：长按即可快速处理
- 副标题：预览详情、切换提醒、归档删除，不用反复进入页面

### 5. 详情与编辑

- 标题：提醒规则和备注都清楚
- 副标题：到期日期、提醒状态、备注信息集中管理，重要细节不遗漏

### 6. 桌面组件

- 标题：不用打开 App 也能看
- 副标题：桌面小组件展示最近到期事项，重要提醒随手可见

## App Store 文案补充

- 副标题建议：到期提醒、归档整理更轻松
- 关键词建议：到期提醒,保质期,证件到期,会员续费,药品提醒,归档整理
- 宣传语建议：把容易忘记的到期事项，变成清晰可处理的日常提醒
